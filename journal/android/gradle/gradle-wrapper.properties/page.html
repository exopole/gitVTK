Ce fichier permet de sélectionner le plugin de gradle utilisé. Ici la version 2.5.
Le Gradle Wrapper est le moyen préféré pour commencer un gradle build. Le wrapper est sous forme d'un script batch sur windows et un script shell pour les autres opérateurs. Quand on commence le build de gradle via le wrapper, Gradle sera automatiquement télécharger et utilisé pour lancer le build.


#FriAug0716:02:05AST2015
distributionBase=GRADLE_USER_HOME
distributionPath=wrapper/dists
zipStoreBase=GRADLE_USER_HOME
zipStorePath=wrapper/dists
distributionUrl=https\://services.gradle.org/distributions/gradle-2.5-all.zip
